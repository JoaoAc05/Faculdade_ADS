Desnvolvimento do Portifólio

Inicialmente tive muita dificuldada para começar o trabalho devido a minha falta de criatividade, fui em busca de muitas referências até que encontrei um portifólio do Matheus Vieira e utilizei como referência principal para o Desenvolvimento do meu trabalho.
No decorrer do trabalho tive varias dificuldades, principalmente na centralização das Sections de projetos e certificados, fiz o trabalho varias vezes não ficando contente com nenhuma paleta de cor que utilizava, refazia novamente em outro formato e com outra paleta até que cheguei neste resultado.
Utilizei muito do CSS para a responsividade do site no maximo de dispositivos de acesso possível. JS foi utilizado em poucas partes onde vi necessidade, nos links do cabeçalho foi utilizado para que a rolagem fosse mais suave, e na escrita de forma progressiva do nome na tela "inicial".

FRAMEWORKS

Os Framework's vem como uma forma mais prática de desenvolver os seus projetos e trabalhos, utilizando ele para automatizar os exercícios repetitivos, como criar um formulário de cadastro, se utiliza o framework pois ele ja tera a requisição e validação de senha e email por exemplo.

REQUISIÇÕES DE API

As requisições de API também vem como auxiliares no desenvolvimento de trabalhos, realizando uma requisição para utilização como banco de dados por exemplo, no site do pokedex que realizei, a requisição da API do pokemon foi utilizada para pegar dados de pokemons para a apresentação no site como nome, imagem, altura e peso.

HTTP

HypeText Transfer Protocol, é o protocolo utilizado pelos navegadores para a exibição de textos compilados como exemplo na linguagem de marcação HTML.

APLICAÇÕES WEB

As Aplicações Web são softwares que estão disponíveis para serem acessados, onde exigem uma entrada (pelo usuário), onde ira realizar um processamento da entrada e exibir uma saída. Sua utilização é para soluções de problema que envolva automações de processos.
